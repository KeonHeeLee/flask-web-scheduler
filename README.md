# flask-web-scheduler
> flask 복습을 위한 To-do 스케줄러 웹 어플리케이션<br>
> 달력에서 자신의 일정과 팀원들의 일정을 관리할 수 있습니다.

## Usage
```bash
python app/app.py
```

- 위와같이 실행을 하면 됩니다.
- ```python 2.7``` 이상의 버전을 권장합니다.
